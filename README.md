# Barbon Media Website

A considered approach to property media — website repository for Netlify deployment.

## Project Structure

```
barbon-media-website/
├── index.html          ← main website (all pages)
├── assets/
│   ├── images/         ← property photos, project images
│   └── fonts/          ← Newsreader + Manrope web fonts (optional)
├── README.md           ← this file
└── brand/
    └── brand-guide.md  ← complete brand reference
```

## Setup for Netlify

1. Connect this repository to Netlify
2. Set build command to: `# (no build needed)`
3. Set publish directory to: `/` (root)
4. Deploy

## Pages & Routes

- `/` — Home (hero, positioning, recent work, disciplines, testimonial, contact CTA)
- `/work` — Work grid with lightbox (3-col masonry, 28+ projects)
- `/services` — Four disciplines explained (Residential, Hospitality, Architecture, Interiors)
- `/studio` — About Barbon (James's note, five principles, Midlands focus)
- `/contact` — Contact form + Spiro booking link + Instagram DM direction

## Brand Assets

All brand guidelines are in `brand/brand-guide.md`. Key reference:

- **Colors:** Bone (#ECE6DA), Ink (#1C1A16), Oxblood (#6B2A2A), Muted (#6E6A60)
- **Fonts:** Newsreader (italic), Manrope (500), JetBrains Mono
- **Tone:** Trusted advisor. Useful before clever. Of the Midlands.
- **Disciplines:** Residential, Hospitality, Architecture, Interiors
